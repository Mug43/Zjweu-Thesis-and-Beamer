## 📊 格式规范实现情况

### 页面设置
- [x] A4 纸张
- [x] 上下边距 28mm
- [x] 左边距 30mm
- [x] 右边距 28mm
- [x] 页眉高度 15pt

### 字体字号
- [x] 正文：小四号宋体，行距22磅
- [x] 章标题：三号宋体加粗居中
- [x] 节标题：小三号宋体加粗居左
- [x] 小节标题：四号宋体加粗居左
- [x] 摘要：小四号宋体，行距20磅
- [x] 关键词标题：小三号宋体加粗
- [x] 图表标题：五号宋体居中
- [x] 参考文献：五号宋体，行距20磅
- [x] 页眉：五号楷体

### 编号规则
- [x] 章：第X章
- [x] 节：X.X
- [x] 小节：X.X.X
- [x] 图：图X-Y
- [x] 表：表X-Y
- [x] 公式：(X-Y)

### 特殊要求
- [x] 段首缩进2字符
- [x] 章节上下空行
- [x] 三线表支持
- [x] 孤行孤尾控制

## 🚀 使用方式

### 快速开始
```bash
# 编译模块化版本（推荐）
xelatex main-example.tex
xelatex main-example.tex

# 编译完整版本
xelatex main.tex
xelatex main.tex
```

### 典型工作流
1. 编辑论文信息（main-example.tex）
2. 编写摘要（Report/abstract.tex）
3. 编写各章节（Report/chp/）
4. 添加图片（Report/Fig/）
5. 编写参考文献（Report/ref/references.tex）
6. 编译生成PDF
