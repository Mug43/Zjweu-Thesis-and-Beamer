## 文件说明

### 1. 核心文件

#### zjweu-thesis-beta.cls（★重要）
LaTeX 模板类文件，包含所有格式定义：
- 页面设置（页边距、纸张大小）
- 字体配置（宋体、楷体、Times New Roman）
- 章节格式（标题样式、编号）
- 图表公式设置（编号规则、样式）
- 摘要环境定义
- 参考文献格式
- 页眉页脚设置

**不建议频繁修改此文件**，除非需要全局格式调整。

#### main.tex
完整版主文档，所有内容（摘要、正文、参考文献）都写在一个文件中。
- 适合简单论文或习惯单文件编写的用户
- 内容和格式耦合度高

#### main-example.tex（★推荐）
模块化主文档，通过 `\input{}` 命令引入各个模块。
- **推荐使用**：结构清晰，便于维护
- 内容和格式分离
- 支持团队协作
- 便于版本控制

#### references.bib
BibTeX 格式的参考文献数据库：
- 包含示例文献条目
- 支持多种文献类型（期刊、图书、会议论文等）
- 可选使用（也可以直接在 references.tex 中编写）

### 2. 文档目录（Report/）

#### abstract.tex
中英文摘要及关键词：
```latex
\begin{cnabstract}
中文摘要内容...
\end{cnabstract}
\cnkeywords{关键词1；关键词2；关键词3}

\begin{enabstract}
English abstract...
\end{enabstract}
\enkeywords{Keyword1; Keyword2; Keyword3}
```

#### chp/ 目录
章节内容，每章一个文件：
- `chapter01.tex` - 第一章（已提供示例）
- `chapter02.tex` - 第二章（需自行创建）
- `chapter03.tex` - 第三章（需自行创建）
- 依此类推...

**章节文件模板**：
```latex
% chapterXX.tex
\chapter{章标题}

\section{节标题}
内容...

\subsection{小节标题}
内容...
```

#### Fig/ 目录
存放论文中使用的所有图片：
- 支持 PNG, JPG, PDF, EPS 等格式
- 推荐使用 PNG 或 PDF 格式（矢量图最佳）

**图片引用示例**：
```latex
\begin{figure}[H]
    \centering
    \includegraphics[width=0.8\textwidth]{Report/Fig/image.png}
    \caption{图片说明}
    \label{fig:image}
\end{figure}
```

#### ref/ 目录
参考文献相关文件：
- `references.tex` - 参考文献内容

## 使用工作流

### 方式一：完整版（main.tex）

1. 直接编辑 `main.tex`
2. 所有内容写在一个文件中
3. 编译：
   ```bash
   xelatex main.tex
   xelatex main.tex
   ```

### 方式二：模块化版（★推荐）

1. **设置论文信息**（main-example.tex）
   ```latex
   \zjweuthesistitle{论文题目}
   \zjweuauthor{姓名}
   \zjweumajor{专业}
   \zjweuadvisor{导师}
   \zjweudate{日期}
   ```

2. **编写摘要**（Report/abstract.tex）

3. **编写各章节**（Report/chp/chapterXX.tex）

4. **添加图片**（Report/Fig/）

5. **编写参考文献**（Report/ref/references.tex 或 references.bib）

6. **在主文件中引入章节**（main-example.tex）
   ```latex
   \input{Report/chp/chapter01}
   \input{Report/chp/chapter02}
   \input{Report/chp/chapter03}
   ```

7. **编译**
   ```bash
   xelatex main-example.tex
   xelatex main-example.tex
   ```

## 常见问题

**Q: 如何添加新章节？**
A: 在 `Report/chp/` 目录创建新文件，然后在 `main-example.tex` 中用 `\input{}` 引入。

**Q: 图片放在哪里？**
A: 统一放在 `Report/Fig/` 目录下。

**Q: 如何切换参考文献方式？**
A: 参见 `参考文献使用说明.md`。

**Q: 编译出错怎么办？**
A: 查看 `.log` 文件，或参考 README.md 中的常见问题部分。
