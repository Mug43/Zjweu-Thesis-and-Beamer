## 文件说明

### 1. 核心文件

#### zjweu-thesis-beta.cls（★重要）
LaTeX 模板类文件 v0.2.0，包含所有格式定义：
- **页面设置**：A4纸张，精确页边距，headsep和footskip控制
- **字体配置**：宋体、楷体、黑体、Times New Roman
- **章节格式**：标题样式、三级编号（章/节/小节）
- **图表公式设置**：自动编号、三线表支持
- **摘要环境**：中英文摘要及关键词
- **参考文献**：国标GB/T 7714格式（gbt7714包）
- **页眉页脚**：自动同步，章节首页同步
- **封面生成**：`\makecover` 命令，支持logo
- **声明页**：`\makedeclaration` 命令
- **致谢环境**：`acknowledgment` 环境
- **防孤行孤字**：自动处理段落孤行

**不建议频繁修改此文件**，除非需要全局格式调整。

#### main-example.tex（★推荐）
模块化主文档，通过 `\input{}` 命令引入各个模块。

**新增配置项**（v0.2.0）：
```latex
% 基本信息
\zjweuthesistitle{论文题目}
\zjweuauthor{姓名}
\zjweumajor{专业}
\zjweuadvisor{导师}
\zjweudate{日期}

% 封面额外信息（新增）
\zjweucollege{学院名称}
\zjweustudentid{学号}
\zjweugrade{届数，如2024}
\zjweucovertitle{封面主标题，如"毕业设计（论文）"}

% 页眉设置（可选）
\zjweuheader{浙江水利水电学院2024届毕业设计（论文）}
```

**新增命令**：
```latex
\makecover        % 生成完整封面（需要logo/logo.png）
\makedeclaration  % 生成声明及授权页

% 致谢环境
\begin{acknowledgment}
致谢内容...
\end{acknowledgment}
```

#### references.bib
BibTeX 格式的参考文献数据库：
- 使用国标GB/T 7714格式
- 支持顺序编码制和著者-出版年制
- 支持多种文献类型（期刊、图书、会议论文等）

### 2. logo 目录（新增）

#### logo.png
学校logo图片，用于封面：
- **尺寸要求**：14.66cm × 4.42cm（约1466×442像素）
- **格式**：PNG（支持透明背景）
- **位置**：封面顶部居中
- **可选**：如不使用 `\makecover`，此文件非必需

### 2. 文档目录（Report/）

#### abstract.tex
中英文摘要及关键词：
```latex
\begin{cnabstract}
中文摘要内容...
\end{cnabstract}
\cnkeywords{关键词1；关键词2；关键词3}

\begin{enabstract}
English abstract...
\end{enabstract}
\enkeywords{Keyword1; Keyword2; Keyword3}
```

#### chp/ 目录
章节内容，每章一个文件：
- `chapter01.tex` - 第一章（已提供示例）
- `chapter02.tex` - 第二章（需自行创建）
- `chapter03.tex` - 第三章（需自行创建）
- 依此类推...

**章节文件模板**：
```latex
% chapterXX.tex
\chapter{章标题}

\section{节标题}
内容...

\subsection{小节标题}
内容...
```

#### Fig/ 目录
存放论文中使用的所有图片：
- 支持 PNG, JPG, PDF, EPS 等格式
- 推荐使用 PNG 或 PDF 格式（矢量图最佳）

**图片引用示例**：
```latex
\begin{figure}[H]
    \centering
    \includegraphics[width=0.8\textwidth]{Report/Fig/image.png}
    \caption{图片说明}
    \label{fig:image}
\end{figure}
```

#### ref/ 目录
参考文献相关文件：
- `references.tex` - 参考文献内容

## 使用工作流

### 方式一：完整版（main.tex）

1. 直接编辑 `main.tex`
2. 所有内容写在一个文件中
3. 编译：
   ```bash
   xelatex main.tex
   xelatex main.tex
   ```

### 方式二：模块化版（★推荐）

1. **设置论文信息**（main-example.tex）
   ```latex
   \zjweuthesistitle{论文题目}
   \zjweuauthor{姓名}
   \zjweumajor{专业}
   \zjweuadvisor{导师}
   \zjweudate{日期}
   ```

2. **编写摘要**（Report/abstract.tex）

3. **编写各章节**（Report/chp/chapterXX.tex）

4. **添加图片**（Report/Fig/）

5. **编写参考文献**（Report/ref/references.tex 或 references.bib）

6. **在主文件中引入章节**（main-example.tex）
   ```latex
   \input{Report/chp/chapter01}
   \input{Report/chp/chapter02}
   \input{Report/chp/chapter03}
   ```

7. **编译**
   ```bash
   xelatex main-example.tex
   xelatex main-example.tex
   ```

## 常见问题

**Q: 如何添加新章节？**
A: 在 `Report/chp/` 目录创建新文件，然后在 `main-example.tex` 中用 `\input{}` 引入。

**Q: 图片放在哪里？**
A: 统一放在 `Report/Fig/` 目录下。

**Q: 如何切换参考文献方式？**
A: 参见 `参考文献使用说明.md`。

**Q: 编译出错怎么办？**
A: 查看 `.log` 文件，或参考 README.md 中的常见问题部分。
