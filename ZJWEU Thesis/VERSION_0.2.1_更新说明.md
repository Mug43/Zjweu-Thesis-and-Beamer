### 2️⃣ 格式修复

#### 🔧 编号格式规范化
**修改前：** 公式、图、表编号格式为 `章节.序号`（如 1.1、图1.1、表1.1）  
**修改后：** 统一改为 `章节-序号`（如 1-1、图1-1、表1-1）

**实现方式：**
```latex
\renewcommand{\theequation}{\thechapter-\arabic{equation}}
\renewcommand{\thefigure}{\thechapter-\arabic{figure}}
\renewcommand{\thetable}{\thechapter-\arabic{table}}
```

## 🎯 使用建议

### 对于新用户
1. 直接使用 `main-example.tex` 作为模板
2. 参考 `chapter01.tex` 和 `chapter02.tex` 学习如何：
   - 插入三线表
   - 插入图片
   - 编写数学公式
   - 引用图表和公式
3. 查看生成的PDF，了解最终效果

### 对于已有用户
1. **如果已在 v0.2.0 基础上编写**：
   - 替换 `zjweu-thesis-beta.cls` 文件
   - 重新编译即可自动应用新的编号格式

2. **如果想参考新示例**：
   - 查看 `chapter02.tex` 获取更多示例代码
   - 复制需要的部分到自己的章节中
