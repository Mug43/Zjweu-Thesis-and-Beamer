# 参考文献使用说明

本模板支持两种参考文献管理方式，您可以根据需要选择其中一种。

## 方式一：使用.bib文件（推荐）

### 优点
- 自动排序和格式化
- 便于管理大量文献
- 支持多篇论文共享文献库
- 避免手动编写格式错误

### 使用步骤

1. **编辑references.bib文件**
   - 在references.bib中添加您的参考文献
   - 使用标准BibTeX格式

2. **修改main.tex**
   - 找到参考文献部分（文件末尾）
   - 注释掉 `\begin{thebibliography}...\end{thebibliography}` 部分
   - 取消以下两行的注释：
   ```latex
   \bibliographystyle{gbt7714-numerical} % 使用国标GB/T 7714数字式
   \bibliography{references} % references.bib是您的参考文献文件名
   ```

3. **编译文档**
   ```powershell
   xelatex main.tex
   bibtex main
   xelatex main.tex
   xelatex main.tex
   ```
   注意：需要运行4次（1次xelatex + 1次bibtex + 2次xelatex）

### BibTeX条目类型示例

**期刊论文**
```bibtex
@article{key,
  author  = {作者},
  title   = {标题},
  journal = {期刊名},
  year    = {年份},
  volume  = {卷},
  number  = {期},
  pages   = {页码}
}
```

**图书**
```bibtex
@book{key,
  author    = {作者},
  title     = {书名},
  publisher = {出版社},
  address   = {出版地},
  year      = {年份}
}
```

**会议论文**
```bibtex
@inproceedings{key,
  author    = {作者},
  title     = {标题},
  booktitle = {会议名称},
  year      = {年份},
  pages     = {页码}
}
```

### 文中引用
```latex
单篇引用：\cite{key}
多篇引用：\cite{key1,key2,key3}
```

## 方式二：直接在文档中编写（当前方式）

### 优点
- 简单直接，无需额外编译步骤
- 适合参考文献较少的情况
- 便于快速查看和修改

### 使用步骤

1. **保持当前代码不变**
   - 使用 `\begin{thebibliography}{99}...\end{thebibliography}` 环境

2. **添加参考文献**
   ```latex
   \bibitem{标签} 作者. 标题[文献类型]. 出版信息, 年份.
   ```

3. **编译文档**
   ```powershell
   xelatex main.tex
   xelatex main.tex
   ```
   注意：只需运行2次xelatex

### 格式示例

**期刊论文 [J]**
```latex
\bibitem{ref1} 张三. 论文标题[J]. 期刊名, 2023, 10(1): 10-15.
```

**图书 [M]**
```latex
\bibitem{ref2} 李四, 王五. 书名[M]. 北京: 出版社, 2024.
```

**学位论文 [D]**
```latex
\bibitem{ref3} 王六. 论文题目[D]. 北京: 清华大学, 2023.
```

**会议论文 [C]**
```latex
\bibitem{ref4} 赵七. 论文标题[C]//会议论文集. 北京: 出版社, 2023: 100-105.
```

**电子文献 [EB/OL]**
```latex
\bibitem{ref5} 作者. 标题[EB/OL]. (发表日期)[引用日期]. URL.
```

### 文中引用
```latex
\cite{ref1}  % 引用单篇
\cite{ref1,ref2,ref3}  % 引用多篇
```

---

## 推荐方案

- **参考文献少于10篇**：使用方式二（直接编写）
- **参考文献超过10篇**：使用方式一（.bib文件）
- **需要投稿多个期刊**：使用方式一（便于切换格式）

## 常见问题

1. **引用编号显示为[?]**
   - 解决：多编译几次，确保引用标签正确

2. **BibTeX编译失败**
   - 检查.bib文件格式是否正确
   - 确保安装了bibtex程序

3. **参考文献格式不符合要求**
   - 方式一：更换bibliographystyle
   - 方式二：手动调整格式
